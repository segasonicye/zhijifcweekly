# 微信公众号模板风格对比

本系统提供**三种风格**的微信公众号模板，您可以根据不同场景选择使用。

---

## 🎨 三种风格对比

### 原版风格（`npm run publish`）

**特点：**
- ✨ 色彩鲜艳，活力四射
- 🌈 渐变背景，视觉冲击力强
- 🎯 适合运动氛围，充满激情

**配色方案：**
- 主标题框：紫色渐变 `#667eea → #764ba2`
- MVP卡片：粉色渐变 `#f093fb → #f5576c`
- 背景色：浅紫渐变 `#f8f9ff → #fff`
- 强调色：蓝色 `#1890ff`

**适合场景：**
- 足球比赛战报
- 充满激情和活力的内容
- 喜欢丰富多彩的视觉效果

**使用命令：**
```bash
npm run publish
```

---

### Ins风格（`npm run publish:ins`）

**特点：**
- ✨ 简洁优雅，现代时尚
- 📐 大量留白，呼吸感强
- 🎓 专业杂志风格，高端大气

**配色方案：**
- 主色：纯黑 `#000`，纯白 `#fff`
- 辅助色：灰色 `#999`, `#666`, `#e8e8e8`
- 点缀色：紫色渐变（仅比分框）`#667eea → #764ba2`
- MVP卡片：橙色渐变 `#ffecd2 → #fcb69f`

**设计元素：**
- 极简边框（1px灰色实线）
- 细线分隔符
- 卡片式布局
- 扁平化设计

**适合场景：**
- 时尚杂志风格内容
- 注重专业感和品质感
- 参考insdaily等高质量公众号

**使用命令：**
```bash
npm run publish:ins
```

---

### 热血外战风格（`npm run publish:battle`）🔥

**特点：**
- 🔥 激情热血，视觉冲击力强
- ⚔️ 战斗氛围浓厚，对抗性强
- 🎯 专为外战设计，点燃战斗激情

**配色方案：**
- 主标题框：红色渐变 `#ff6b6b → #ee5a24`
- MVP卡片：金色渐变 `#ffd93d → #ff9500`
- 出勤名单：深色背景 `#2d3436 → #000000`
- 强调色：高对比红、黑、金

**设计元素：**
- 战斗符号（剑⚔️、盾🛡️、奖杯🏆）
- 照片自动编号标记
- 深色背景+金色文字
- 动感装饰元素
- 强烈的视觉对比

**适合场景：**
- 外战对抗赛 ⭐ 推荐
- 重要比赛
- 需要营造战斗氛围
- 激烈对抗的战报

**使用命令：**
```bash
npm run publish:battle
```

## 🎯 使用建议

### 选择原版风格，如果：
- ✅ 通用足球比赛战报
- ✅ 希望营造热烈的运动氛围
- ✅ 喜欢丰富多彩的视觉效果
- ✅ 受众喜欢活力风格

- ✅ **内战**训练赛 ⭐ 推荐
- ✅ 追求时尚杂志的品质感
- ✅ 希望更专业、更简洁
- ✅ 参考insdaily等高质量公众号
- ✅ 注重留白和阅读舒适度

### 选择热血外战风格，如果：
- ✅ **外战**对抗赛 ⭐ 推荐
- ✅ 重要比赛或决赛
- ✅ 需要营造战斗氛围
- ✅ 激烈对抗的战报
- ✅ 想要点燃激情

## 🔄 快速切换

**生成原版风格：**
```bash
npm run publish
# 输出: output/wechat-YYYY-MM-DD-对手.html
```

**生成Ins风格（适合内战）：**
```bash
npm run publish:ins
# 输出: output/wechat-YYYY-MM-DD-对手-ins.html
```

**生成热血外战风格（适合外战）：**
```bash
npm run publish:battle
# 输出: output/wechat-YYYY-MM-DD-对手-battle.html
```

三个版本的输出文件不同，可以同时存在，方便对比选择。

## 💡 自定义建议

如果您想调整风格，可以修改以下文件：

**原版风格：**
- `scripts/wechat-template.js` - 模板文件
- `scripts/publish-wechat.js` - 发布脚本

**Ins风格：**
- `scripts/wechat-template-ins.js` - 模板文件
- `scripts/publish-ins.js` - 发布脚本

**热血外战风格：**
- `scripts/wechat-template-battle.js` - 模板文件
- `scripts/publish-battle.js` - 发布脚本

### 常见修改：

**修改配色：**
找到对应的颜色值并替换：
```javascript
// 例如：紫色渐变
background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);

// 改为蓝色渐变
background: linear-gradient(135deg, #4facfe 0%, #00f2fe 100%);
```

**调整间距：**
修改 `padding` 和 `margin` 值：
```javascript
// 增加留白
padding: 40px;  // 原30px
margin: 50px 0; // 原35px 0
```

**调整字体：**
```javascript
font-size: 18px;  // 调整文字大小
font-weight: 600; // 调整文字粗细
letter-spacing: 1px; // 调整字间距
```

## 🚀 开始使用

1. **查看原版风格：**
   ```bash
   npm run publish
   ```

2. **查看Ins风格（适合内战）：**
   ```bash
   npm run publish:ins
   ```

3. **查看热血外战风格（适合外战）：**
   ```bash
   npm run publish:battle
   ```

4. **在浏览器中打开预览文件对比：**
   - `output/wechat-YYYY-MM-DD-对手.html`
   - `output/wechat-YYYY-MM-DD-对手-ins.html`
   - `output/wechat-YYYY-MM-DD-对手-battle.html`

5. **选择喜欢的风格复制到公众号：**
   - 浏览器中 Ctrl+A 全选
   - Ctrl+C 复制
   - 公众号编辑器 Ctrl+V 粘贴
