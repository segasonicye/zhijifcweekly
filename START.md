# 🎉 系统已完成!

## 快速开始

### 1. 创建新战报
```bash
npm run new
```

### 2. 编辑战报
打开 `matches/` 目录下生成的文件,参考示例 `matches/2025-01-12-内战.md` 编写

### 3. 添加照片
```bash
npm run photos
```

### 4. 查看统计
```bash
npm run stats
```

### 5. 发布到公众号
```bash
npm run sync
```
