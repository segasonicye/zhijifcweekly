# 快速开始指南

## 第一次使用

### 1. 安装依赖

```bash
npm install
```

### 2. 创建第一篇战报

```bash
npm run new
```

输入示例:
```
请输入比赛日期 (YYYY-MM-DD): 2025-01-12
请输入对手名称: 星光联队
请输入比分 (如 3-2): 3-2
请输入比赛地点: 朝阳公园
```

### 3. 编写战报

打开生成的文件 `matches/2025-01-12-星光联队.md`,参考下面格式编写:

```markdown
---
title: "逆转取胜!河伯FC 3-2 战胜星光联队"
date: 2025-01-12
opponent: 星光联队
score: "3-2"
location: 朝阳公园
scorers:
  - name: 喜力授
    minute: 12
    assist: 德国小弟
  - name: 王五
    minute: 78
attendance: [辉哥, 托蒂, 王书记, 喜力授, 叶伯海, 德国小弟, 张航, 小王, 东哥]
photos:
  - path: "photos/2025-01-12/photo-001.jpg"
    caption: "赛后合影"
---

## 比赛回顾

冬日不虚度,晴光忽满帘,周六上午知己队再战福沁球场...

![赛后合影](photos/2025-01-12/photo-001.jpg)

### 精彩瞬间

第12分钟,德国小弟助攻喜力授破门...
```

### 4. 添加照片

将照片放到临时文件夹(如桌面/temp_photos),然后:

```bash
npm run photos
```

输入:
```
请输入照片所在的目录路径: C:\Users\segas\Desktop\temp_photos
请输入比赛日期 (格式: YYYY-MM-DD): 2025-01-12
```

照片会自动复制到 `photos/2025-01-12/` 并重命名。

### 5. 查看统计数据

```bash
npm run stats
```

会显示:
- 出勤排行榜
- 进球排行榜
- 助攻排行榜

### 6. 发布到公众号

```bash
npm run sync
```

选择要发布的比赛序号,生成的HTML文件在 `output/` 目录。

## 每周使用流程

```
1. 踢球回来
   ↓
2. npm run new (创建战报模板)
   ↓
3. 编写比赛内容 (matches/目录下)
   ↓
4. npm run photos (整理照片)
   ↓
5. npm run stats (查看统计数据)
   ↓
6. npm run sync (转换为公众号格式)
   ↓
7. 复制到公众号发布
   ↓
8. git add/commit (保存记录)
```
